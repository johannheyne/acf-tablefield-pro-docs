The JSON string has multiple sections and begins with an object `"p"` which stands for parameters.

## Table Parameter

This parameter object has an object `"o"` which stands for options.

### Table Options

| Name           | Type           | Decription     |
| :------------- | :------------- | :------------- |
| uh             | boolean        | Use header     |
