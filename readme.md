# Readme

### Start Jekyll
```
# terminal
bundle exec jekyll serve
```
[http://localhost:4000/](http://localhost:4000/)

----
### updates gem manager
```
# terminal
gem update --system
```

### list gems, list outdated gems, updates gems
```
# terminal
gem list
gem outdated
gem update
```

### rebuild snapchot from scratch using only the gems in Gemfile
```
# terminal
bundle update
```
